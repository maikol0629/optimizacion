# Curso de Optimización

Este repositorio contiene un conjunto de implementaciones (en formato de libretas de Jupyter) de los algoritmos y métodos numéricos vistos a la largo del curso de Optimización. Estos algoritmos son fundamentales para resolver problemas de búsqueda de raíces y optimización (maximización y minimización) de funciones unidimensionales

El código fuente de cada técnica se encuentra en la carpeta `métodos_programados`.

## Requisitos de Ejecución

Para poder ejecutar estas celdas y probar los métodos por tu cuenta se recomienda contar con:
- Un entorno compatible con Jupyter Notebook o JupyterLab.
- Python 3.x
- Bibliotecas fundamentales para resolución numérica y graficación como `numpy`, `sympy` (si aplica) y `matplotlib`.
